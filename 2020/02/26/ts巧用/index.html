五. 辨别 type & interface

相同点：都可以描述一个对象或者函数
不同点：interface 可以 extends， type 是不允许 extends 和 implement 的，
但是 type 缺可以通过交叉类型 实现 interface 的 extend 行为，并且两者并不是相互独立的，也就是说 interface 可以 extends type, type 也可以 与 interface 类型 交叉 。
